About

Hi! My name is Meghan and this website is a part of my senior thesis project for the BFA Graphic Design program at Virginia Commonwealth University. It's ultimately meant to be a conversation starter surrounding the concept of alt text and image description.

I am particularly interested in / passionate about digital accessibility and as I've gained some experience with digital design, I've noticed how image description is not always the easiest thing to do - especially when you're first starting out. It's a skill that requires learning and practice.

I ultimately want this website to be a place where people can learn and practice - and hopefully have fun while doing that.

I've included a couple of things in this website. I've first included a summary of information that I've encountered about alt text / image descriptions, doing my best to both define them and give examples of what those can look like. This is the Introduction to alt text section.

After that, I've included an outline of a workshop that you can do on your own time (either by yourself or with others), if you want to get some hands on (and I think kind of fun) practice with alt text and image descriptions. This is the Workshop page.

I've also included a place where you can practice here on this website with pictures I found in magazines. This is the Practice page.

Lastly, I've shared my sources. This is the Resource Library page.
