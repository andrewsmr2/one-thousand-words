If you were writing alt text or an image description of the photo below, what would you write? It's okay if this feels clumsy at first, that's normal!

Some thoughts you can start off with:

  • What's in the photo? List what you see.
  • What are some words that come to mind when you see this photo?
  • How would you describe this photo to a friend? A stranger?
  • I need to make sure I'm not fully copying alt text as poetry here
